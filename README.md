# Hazmat Database Expansion Project

## 專案介紹 (Project Overview)
本專案為 HAZMAT (危險物質) 快速應變查詢網頁，旨在為第一線消防及化災應變人員提供直覺、準確且即時的危害辨識與戰術引導資訊。
資料庫已全面從原先約 400 筆核心物質，擴充至完整涵蓋 **2020 應變指南 (ERG 2020)** 共 **2,863 筆** 化學物質與混合物，並完美整合了高價值的黃金欄位資訊與 UI 體驗。

## 最新更新與功能優化
本專案進行了全面的資料整合與架構修復，包含以下重點：

1. **黃金核心資料智能合併**
   - 將原先約 400 筆精修物質的「黃金屬性」（包含精準 Hazard Class 分類、NFPA 704 四色菱形圖示、LaTeX 化學反應方程式、客製化防護衣與隔離戰術建議）完美融入 2,863 筆的全量資料庫中，解決了因擴充造成的資料流失。

2. **繁體中文官方名冊校對**
   - 全面比對並對照台灣官方化學物質中文名冊進行了校對與修正。
   - 保留原有的俗名（例如：重油/輕油、柴油）與英文名稱，支持多語系、俗名、UN 編號及 CAS 號碼的全方位模糊搜索。

3. **實用消防應變戰術引導**
   - 徹底移除了所有「請參閱 ERG 指南第 000 頁」的無效預設描述。
   - 根據 ERG 指導編號、TIH (吸入性危害) 狀態及禁水 (WR) 特質，智慧生成了精確的滅火與禁忌注意事項（例如：抗醇型泡沫、一般泡沫、乾粉、二氧化碳，以及對禁水物質的嚴禁用水警告）。

4. **TCS 毒性化學物質查詢與跳轉優化**
   - 移除了 260 餘筆通用運送類別 (N.O.S) 物質的無效 `isTCS` (毒物) 標記，避免在線上查詢時跳轉出錯。
   - 針對如 `[UN 3318] 氨溶液` 等 30 筆列管具體溶液補齊了主體 CAS，並在跳轉查詢時過濾括號尾碼（如 `(COCl2)`），保證 100% 能在台灣官方毒性及關注化學物質登記系統中查到列管資訊。

5. **無 CAS 資訊物質之中文 SDS 搜尋補強**
   - 對於無 CAS 號碼的混合物或通用物質，前端自動提取中英文名稱並生成「🇹🇼 中文 SDS 搜尋 (依名稱)」按鈕，讓一線人員能在沒有 CAS 碼時依然能一鍵搜出中文安全資料表。

6. **視覺體驗與 UI 強化**
   - **GHS 多圖標修復**：修正 GHS 在前端只顯示單一圖標的 Bug，可正確渲染多個 GHS 危害圖標。
   - **無 NFPA 704 數據提示**：若該化學品無 NFPA 704 資料，系統會自動隱藏空菱形圖，改以精緻的提示框呈現，大幅提升整體網頁的視覺協調。
   - **Class N/A 破圖修復**：對於 Class 類別為 N/A 的物質，前端將改為渲染 CSS 菱形問號 `❓` 標誌，避免加載不存在的圖片而產生破圖。

7. **未知物質直接網頁查詢功能**
   - 當搜尋結果為空（查無相符結果）時，會顯示快速網頁查詢按鈕，一鍵跳轉 Google 搜尋該未知物質的 SDS 安全資料表。


## 檔案說明
- `index.html`：網頁主程式，包含搜尋介面、詳細資料卡片及免責聲明。
- `database.js`：收錄 2,863 筆化學物質的完整 JSON 資料庫。
- `README.md`：專案說明文件。

## 網站使用
資料庫儲存於 `database.js`，可以直接與 `index.html` 一同於本地或靜態伺服器中運行。
