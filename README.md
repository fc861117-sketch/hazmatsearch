# Hazmat Database Expansion Project

## 專案介紹 (Project Overview)
本專案為 HAZMAT (危險物質) 快速應變查詢網頁，資料庫從原本約 400 筆擴充至涵蓋 **2020 應變指南 (ERG 2020)** 所有的 2,863 筆化學物質與混合物。

## 最新更新：全量資料庫升級與優化
本次資料庫升級進行了以下重點處理：
1. **全量擴充**：成功將 2,863 筆化學物質匯入資料庫。
2. **自動中文翻譯**：全面使用機器翻譯將英文化學名轉譯為繁體中文，方便中英雙語及俗名搜尋 (例如：丙酮, Acetone)。
3. **CAS 號碼權威對比**：針對 ERG 內所有化學名稱，利用正則表達式清除附註 (如 solution, stabilized 等)，並送交 **PubChem 權威 API** 進行校對，共成功找回 1,265 種精確單一物質的 CAS 號碼。剩餘的通用類別 (N.O.S) 與混合物則自動免除驗證以避免報錯。
4. **精確標示**：保留並標記 TIH (吸入性危害) 與 WR (禁水性) 的物質特徵，針對高危險物質加入了預設的戰術與防護衣建議 (例如：A級氣密衣、100m 隔離區)。
5. **高危險物質特例**：針對鹼金屬、碳化物等禁水物質，加入了詳細的 LaTeX 化學反應方程式。
6. **高品質黃金欄位合併與修復 (2026/06/04)**：
   - **黃金資料智能恢復**：將原先 ~400 筆化學物質的黃金精修屬性（如 Class 類別、NFPA 704 四色菱形圖、詳細滅火與禁忌注意事項、化學反應式 LaTeX）智能合併回 2,863 筆完整大資料庫，解決了因擴充造成的資料流失。
   - **GHS 標示與 TIH 危害圖標修復**：修正 GHS 在前端只顯示單一圖標的 Bug（改為符合預期的單一逗號分隔陣列），並修復 TIH (吸入性毒性) 在前端的 `☠️TIH` 與 `🈲禁水` 危害標籤顯示。

## 網站使用
資料庫儲存於 `database.js`，可以直接與 `index.html` 一同於本地或靜態伺服器中運行。
